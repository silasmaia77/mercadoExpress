# 🤖 Agente de IA para Atendimento via WhatsApp

## 📌 Descrição

Este projeto consiste em um agente de Inteligência Artificial desenvolvido para automatizar o atendimento ao cliente via WhatsApp, utilizando IA generativa para fornecer respostas rápidas, naturais e eficientes.

A solução foi projetada para atender empresas que desejam melhorar a comunicação com clientes e reduzir o tempo de resposta.

## 🎯 Problema que resolve
Empresas frequentemente enfrentam:
- Alto volume de mensagens
- Demora no atendimento
- Sobrecarga da equipe

Este agente automatiza o atendimento, garantindo agilidade e disponibilidade 24/7.

## ⚙️ Funcionalidades
- Atendimento automatizado via WhatsApp  
- Respostas inteligentes com IA  
- Simulação de atendimento humano  
- Integração com API de mensageria (WAHA)  
- Fluxos personalizados conforme o negócio  

## 🧰 Tecnologias utilizadas
- Inteligência Artificial (ChatGPT / LLM)  
- Python  
- Docker  
- API WhatsApp (WAHA)  
- Visual Studio Code
